# AI Wing Design Program 사용자 설명서

## 1. 프로그램 실행

기본 실행 파일은 `WingDesignerAI.exe`입니다.

1. `WingDesignerAI.exe` 더블클릭

실행이 되면 데스크톱 창이 열리고 메인 화면이 표시됩니다.

참고:

- `Run_WingDesignerAI.bat`는 문제 진단용 보조 실행 스크립트입니다.
- 일반 사용자는 `WingDesignerAI.exe`만 사용하면 됩니다.

## 2. 시작 전 설정 (AI API 키)

AI 설계 기능을 사용하려면 API 키가 필요합니다.

### 방법 A: 프로그램 내부에서 설정 (권장)

1. 프로그램 상단의 `AI 설정` 버튼 클릭
2. 사용하는 제공자 키 입력
   - Gemini
   - GPT (OpenAI)
   - Claude (Anthropic)
   - Grok (xAI)
3. 제공자와 모델 선택
4. `설정 저장` 클릭

### 방법 B: `.env` 파일 직접 편집

`AI_Design_Program_Run` 폴더의 `.env` 파일을 열어 아래와 같이 입력합니다.

```env
GEMINI_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
ANTHROPIC_API_KEY=your_key_here
XAI_API_KEY=your_key_here
```

## 3. 기본 사용 절차

1. 왼쪽 채팅창에 원하는 설계 조건 입력
   - 예: 기체 종류, 날개 폭, 목표 비행 속도, 안정성 우선 여부
2. AI 질문에 답하면서 설계 조건 구체화
3. 설계가 완료되면 오른쪽 탭에서 결과 확인
   - `3D 모델`: 형상 확인, STL/OBJ 내보내기
   - `에어포일`: 단면 형상 확인
   - `제원`: 크기/각도/공력 데이터 확인
4. 필요하면 추가 대화로 설계 수정
5. 상단 `히스토리` 또는 `이름 저장`으로 버전 저장

## 4. 히스토리 기능

- `히스토리` 버튼: 저장된 설계 목록 열기
- `현재 상태 저장`: 현재 설계를 새 항목으로 저장
- `선택 2개 비교`: 두 설계의 제원 비교
- 항목 불러오기: 저장된 설계를 현재 화면으로 복원

## 5. 파일 저장 위치

실행 중 생성되는 데이터는 `data` 폴더에 저장됩니다.

- `data/session_data.json`: 최근 작업 세션
- `data/history_data.json`: 히스토리 목록
- `data/provider_settings.json`: API/모델 설정

중요:

- 기록을 유지하려면 `data` 폴더를 삭제하지 마세요.
- 다른 PC로 옮길 때는 프로그램 폴더 전체를 함께 복사하세요.

## 6. 자주 발생하는 문제

### 프로그램이 실행되지 않음

- 먼저 `WingDesignerAI.exe`를 관리자 권한으로 한 번 실행해 보세요.
- 보안 프로그램이 실행을 차단했는지 확인하세요.
- 경로에 한글/특수문자가 많을 경우 다른 폴더로 옮겨 재시도하세요.
- 계속 실행이 안 될 때만 `Run_WingDesignerAI.bat`로 상세 오류를 확인하세요.

### AI가 응답하지 않음

- API 키 입력 여부 확인
- `AI 설정`에서 선택한 제공자/모델 확인
- 인터넷 연결 상태 확인

### 히스토리/설정이 사라짐

- `data` 폴더가 초기화되지 않았는지 확인
- 백업에서 `data` 폴더를 복원

## 7. 권장 사용 팁

- 처음에는 목표를 간단히 입력한 뒤 AI 질문에 답하며 좁혀가면 결과가 안정적입니다.
- 설계가 마음에 들면 바로 `이름 저장`으로 버전을 남기세요.
- 3D 출력이 필요하면 STL/OBJ를 내보내고 별도 CAD/슬라이서에서 후처리하세요.
